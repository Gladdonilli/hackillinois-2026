**Frozen**: 2026-02-28
**Status**: LOCKED — all agents must conform to this contract.

## 1. API Envelope

ALL endpoints return:

```typescript
interface ApiResponse<T> {
  success: boolean
  data?: T
  error?: { code: string; message: string }
}
```

### Error Codes

| Code                | HTTP | Meaning                        |
| ------------------- | ---- | ------------------------------ |
| `UPLOAD_TOO_LARGE`  | 413  | File exceeds 10MB              |
| `INVALID_FORMAT`    | 400  | Not .wav/.mp3/.flac/.ogg       |
| `PROCESSING_FAILED` | 500  | Pipeline error                 |
| `MODEL_UNAVAILABLE` | 503  | Modal/GPU not ready            |
| `RATE_LIMITED`       | 429  | Exceeds 20 req/min             |
| `NOT_FOUND`         | 404  | Report ID doesn't exist        |

## 2. Endpoints

### POST /api/analyze

**Input**: `multipart/form-data`, field=`file`, max 10MB, formats: wav/mp3/flac/ogg

**Response**: SSE stream (`text/event-stream`)

**Headers**:
- `X-Request-ID`: UUID (generated by Worker)
- `X-Processing-Time-Ms`: total ms (on verdict)
- CORS: `Access-Control-Allow-Origin` from allowlist

### GET /api/reports/:reportId

**Response**: `ApiResponse<Report>`

```typescript
interface Report {
  reportId: string
  createdAt: string        // ISO 8601
  verdict: 'GENUINE' | 'DEEPFAKE'
  confidence: number       // 0-1
  peakVelocity: number     // cm/s
  threshold: number        // cm/s
  anomalousFrameCount: number
  totalFrameCount: number
  anomalyRatio: number     // 0-1
  audioKey: string         // R2 key
  durationSeconds: number
  processingTimeMs: number
  // Hybrid v1 (optional, present when classifier ran)
  classifierScore?: number   // 0-1 probability of genuine
  classifierModel?: string   // e.g. "gbm_252feat_v1"
  ensembleScore?: number     // weighted fusion of formant + classifier
}
```

### GET /api/health

**Response**: `ApiResponse<{ service: "larynx", status: "healthy", timestamp: number }>`

### GET /api/history

**Response**: `ApiResponse<Report[]>`

Returns the 50 most recent analysis reports, ordered by `created_at` descending.
Used by the frontend history panel. Not authenticated — scoped to all users.

All SSE events follow: `event: <type>\ndata: <json>\n\n`

```json
{
  "step": "loading_audio | extracting_formants | mapping_articulatory | computing_velocity | running_classifier",
  "progress": 0.0-1.0,

```

## 4. D1 Schema (analysis_reports)

```sql
CREATE TABLE IF NOT EXISTS analysis_reports (
  report_id    TEXT PRIMARY KEY,
  created_at   TEXT NOT NULL DEFAULT (datetime('now')),
  audio_key    TEXT,              -- R2 object key
  duration_s   REAL,
  verdict      TEXT NOT NULL,     -- 'GENUINE' or 'DEEPFAKE'
  confidence   REAL NOT NULL,
  peak_velocity REAL NOT NULL,
  threshold    REAL NOT NULL,
  anomalous_frames INTEGER NOT NULL,
  total_frames INTEGER NOT NULL,
  anomaly_ratio REAL NOT NULL,
  processing_time_ms INTEGER,
  classifier_score REAL,          -- hybrid v1
  classifier_model TEXT,          -- hybrid v1
  ensemble_score REAL,            -- hybrid v1
  client_ip_hash TEXT
);

CREATE INDEX IF NOT EXISTS idx_reports_created ON analysis_reports(created_at);
```
