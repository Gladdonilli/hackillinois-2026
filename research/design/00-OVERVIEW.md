## Core Thesis

**"Clinical Authority + Earned Spectacle"** — 90% monochrome silence. Color, motion, and sound are EARNED by the data. The violation hits 10x harder because everything else is quiet.

## Strategic Recommendation

**Hybrid: Clinical Baseline + Fluoroscopy Violation + Geiger Sound**

- Rest state: Clean dark instrument panel (zinc monochrome, sharp corners, high data density)
- Violation moment: Skull shifts to green-phosphor scan, tongue clip zone ignites thermal heatmap
- Sound: Geiger velocity clicks create unconscious dread before visual payoff

## Non-Negotiable Principles (All Directions)

1. Color is SCARCE — 90% monochrome, color = data state only
2. Silence is baseline — sound only when data demands it
3. Violation is 0ms — no easing, no transition, physics doesn't animate
4. Typography rupture — number breaks its container, 400% size jump
5. 400ms freeze before reveal — let audience notice something is about to change
6. ALL other UI desaturates during violation — only violation retains color
7. 60fps or death — cut any effect that drops frames
8. Ghost trail — show where tongue SHOULD be (gray) vs where it IS (red)
